Сервисы обмена сообщений

E-mail

Постоянные e-mail и алиасы

msgsafe.io (постоянный e-mail; забанен в России, для доступа нужно использовать TOR или VPN; до 10 бесплатных алиасов (удалённые алиасы тоже считаются); возможно зарегистрироваться через TOR; для того, чтобы отсылать сообщения, нужно указать другой e-mail, например mail2tor; Discord не позволяет подтверждать свои аккаунты с этим e-mail, некоторые игры - тоже) mail2tor.com (почтовый onion-сервис. Присутствие кириллицы в заголовке e-mail'a приводит к невозможности прочитать текст сообщения. Присутствие кириллицы в тексте сообщения приводит к тому, что сообщение возможно прочитать только скачав его в html-формате) Псевдонимы для адресов e-mail (требуется аккаунт на accounts.firefox.com; обратите внимание, это сервис от фирмы Mozilla, доверие к ней крайне ограниченное [в чёрном списке сайта], но в бытовом плане пользоваться можно)

Криптография и стеганография

VeraCrypt (шифрование дисков [Windows, Linux]) GnuPG для Windows (не самый стойкий продукт, по-умолчанию использует 128-битное шифрование). Под Linux консольный GPG (GnuPG) уже входит в установку по умолчанию. Можно также работать с репозиториями gpa и kleopatra. Под Linux также можно использовать OpenSSL (см. инструкции в курсе 1, 2). KeePass (менеджер паролей) (под Mint можно попробовать использовать репозитории keepass2 или keepassx) 7-zip (архиватор; есть встроенные функции шифрования; использование для шифрования незначительной информации; 7z установлен по-умолчанию в Linux) OpenPuff (стеганографическое приложение) MultiObfuscator (приложение для шифрования файлов; не доверяйте этой программе слишком сильно; лучше ей шифровать уже зашифрованные данные) BlackDisplay. Программа для шифрования и уничтожения данных для Windows, требует .NET. Не рекомендуется к использованию, т.к. снята с поддержки и немного глючная. После первого запуска в настройках поставьте флажок "отключить напоминания по умолчанию" и режим "не беспокоить". Выйдите из программы и измените в файле "options.txt" (в директории программы) строку "bool :35LogEnabled:Да" на строку "bool :35LogEnabled:Нет" - это полностью отключит напоминания о времени отдыха и логирование вашей активности. Также можете включить опцию "при старте программы сразу же начинать собирать данные для генерации паролей". Если вы знаете хорошие бесплатные приложения для шифрования, напишите мне.

Диски в оперативной памяти (без шифрования)

Для Windows: ImDisk (ImDisk Toolkit) Dataram RamDisk (смотрите бесплатную версию - один диск до гигабайта) Для Linux: Используйте стандартный tmpfs (без возможности сохранить на диск; для сохранения - используйте тома VeraCrypt) Пример записи в файле /etc/fstab none /dirInRam tmpfs noexec,rw,nosuid,nodev,size=512M (noexec - запрет запуска, size=512M - максимум 512 Мб в памяти)

Удаление данных

Для Windows: Eraser программа для удаления файлов и очистки свободного места на диске. Делает примерно то же, что и BlackDisplay (см. выше в криптографии), но требует административных привилегий и уничтожает более надёжно и на более низком уровне (рекомендуется). Под Linux можно использовать shred или srm Примеры shred -n 0 -zvu -f txt.txt shred -n 0 -zvu -f * Команда уже есть в Linux .NET или Mono: Программа для перезатирания для .NET (работает с Mono под Linux) [бинарники]
