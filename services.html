На главную

Общие сетевые сервисы:

  • TorBrowser (скачивание через TOR)
  • локальный прокси-сервер privoxy со списками доверия trust.txt
  • виртуальная машина VirtualBox, на которую можно устанавливать различные ОС, например: Ubuntu, Tails, Mint, Fedora, Whonix
  • Docker - программа для контейнеризации процессов в ОС Linux и Windows 10, позволяющий изолировать процессы от других процессов в ОС без виртуальной машины
  • Анонимный децентрализованный torrent-клиент

Сервисы обмена сообщений

E-mail Постоянные e-mail и алиасы tuta.io она же tutanota.com (постоянный e-mail; при попытке создать ящик через TOR банит на 48-мь часов; однако, после этого срока, ящиком возможно пользоваться) msgsafe.io (постоянный e-mail только на приём сообщений; до 10 бесплатных алиасов; возможно зарегистрироваться через TOR; для того, чтобы отсылать сообщения, нужно указать другой e-mail) Псевдонимы для адресов e-mail (требуется аккаунт на accounts.firefox.com; обратите внимание, это сервис от фирмы Mozilla, доверие к ней крайне ограниченное [в чёрном списке сайта], но в бытовом плане пользоваться можно) Protonmail (e-mail с функциями самоуничтожения; забанен в России, для доступа нужно использовать TOR или VPN; не позволяет создавать через TOR, требует указания другого e-mail, за временные e-mail банит; имеет функции автоматического удаления сообщений по прошествию времени) Временные e-mail temporary-email-address.com (одноразовый e-mail - требует JavaScript) anonymousemail.me (отсылка e-mail без создания ящика; в начале письма вставляет предупреждающий текст) emailondeck.com guerrillamail.com mailnesia.com temp-mail.org (одноразовые e-mail - требуют JavaScript; невозможно использовать из-под TOR; много доменных имён, поэтому если один ящик не прошёл, можно попробовать создать другой) Иные сервисы обмена сообщений privnote.com (сервис самоуничтожающихся записок; требуется JavaScript - это плохо, т.к. может использоваться для отслеживания; однако, заявляется, что шифрование записки происходит на клиенте - для этого и нужен JavaScript) onetimesecret.com (ещё один сервис самоуничтожающихся записок; не требует JavaScript) Русский сервис одноразовых записок в TOR (требует JavaScript) bitmessage.org (анонимный децентрализованный чат-клиент)

Криптография и стеганография

VeraCrypt (шифрование дисков) OpenSSL для Windows GnuPG для Windows (не самый стойкий продукт, по-умолчанию использует 128-битное шифрование) KeePass (менеджер паролей) 7-zip (архиватор; есть встроенные функции шифрования; использование для шифрования незначительной информации) OpenPuff (стеганографическое приложение) MultiObfuscator (приложение для шифрования файлов; не доверяйте этой программе слишком сильно; лучше ей шифровать уже зашифрованные данные) BlackDisplay. Программа для шифрования и уничтожения данных для Windows, требует .NET. Не рекомендуется к использованию, т.к. снята с поддержки и немного глючная. После первого запуска в настройках поставьте флажок "отключить напоминания по умолчанию" и режим "не беспокоить". Выйдите из программы и измените в файле "options.txt" (в директории программы) строку "bool :35LogEnabled:Да" на строку "bool :35LogEnabled:Нет" - это полностью отключит напоминания о времени отдыха и логирование вашей активности. Также можете включить опцию "при старте программы сразу же начинать собирать данные для генерации паролей". Если вы знаете хорошие бесплатные приложения для шифрования, напишите мне.

Удаление данных

Eraser программа для удаления файлов и очистки свободного места на диске. Делает примерно то же, что и BlackDisplay (см. выше в криптографии), но требует административных привилегий и уничтожает более надёжно и на более низком уровне (рекомендуется).
